# Patch 16 — llm.sw:854

## Problem
`extract_content_impl` calls `length(choices)` without checking whether `map_get(decoded, 'choices')` returned nil, so a malformed API response that omits the `choices` key crashes the process.

## Current code
```sw
            choices = map_get(decoded, 'choices')
            if (length(choices) == 0) { nil }
            else {
                choice0 = hd(choices)
                msg_obj = map_get(choice0, 'message')
                map_get(msg_obj, 'content')
            }
```

## Proposed fix
```sw
            choices = map_get(decoded, 'choices')
            if (choices == nil) { nil }
            else { if (length(choices) == 0) { nil }
            else {
                choice0 = hd(choices)
                msg_obj = map_get(choice0, 'message')
                map_get(msg_obj, 'content')
            }}
```
