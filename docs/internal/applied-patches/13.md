# Patch 13 — log.sw:166

## Problem
`tail_recent()` splices the telemetry file path into a shell command without quoting, breaking on spaces in `$HOME` and allowing shell injection.

## Current code
```sw
fun tail_recent(n) {
    p = path()
    if (file_exists(p) == 'false') {
        "(no telemetry for today yet)"
    } else {
        cmd = "tail -n " ++ to_string(n) ++ " " ++ p ++ " 2>&1"
        r = shell(cmd)
        out = elem(r, 1)
        if (string_length(out) == 0) { "(empty)" } else { out }
    }
}
```

## Proposed fix
```sw
import Util

fun tail_recent(n) {
    p = path()
    if (file_exists(p) == 'false') {
        "(no telemetry for today yet)"
    } else {
        cmd = "tail -n " ++ to_string(n) ++ " " ++ Util.shell_q(p) ++ " 2>&1"
        r = shell(cmd)
        out = elem(r, 1)
        if (string_length(out) == 0) { "(empty)" } else { out }
    }
}
```
