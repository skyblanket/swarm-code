# Patch 20 — arthopod.sw:201

## Problem
`generate_soul` concatenates `getenv("HOME")` without checking for nil, causing a runtime crash when HOME is unset.

## Current code
```sw
    # Persist.
    file_mkdir(getenv("HOME") ++ "/.swarm-code")
    file_write(soul_path, json_encode(final_soul))
    final_soul
```

## Proposed fix
```sw
    # Persist.
    home = getenv("HOME")
    dir = if (home == nil) { "" } else { home }
    file_mkdir(dir ++ "/.swarm-code")
    file_write(soul_path, json_encode(final_soul))
    final_soul
```

safe

## Confidence
high — follows existing `if (x == nil) { ... } else { ... }` idiom used on line 162 of the same file.
