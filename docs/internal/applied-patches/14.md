# Patch 14 — log.sw:182

## Problem
`summarize()` interpolates the telemetry path `p` directly into a shell command without quoting, causing command injection / breakage when the path contains spaces (e.g. `$HOME` with spaces).

## Current code
```sw

    if (file_exists(p) == 'false') {
        "(no telemetry for today yet)"
    } else {
        cmd =

            "echo '  --------------------------'; " ++
            "printf '  sessions     : '; grep -c '\"session_start\"' " ++ p ++ "; " ++
            "printf '  user inputs  : '; grep -c '\"user_input\"' " ++ p ++ "; " ++

            "echo; " ++
            "echo '  most-used tools:'; " ++

            "echo '  recent errors:'; " ++

        elem(r, 1)
```

## Proposed fix
```sw

    p = path()
    if (file_exists(p) == 'false') {
        "(no telemetry for today yet)"
    } else {
        cmd =
            "echo '  session summary for today'; " ++
            "echo '  --------------------------'; " ++
            "printf '  sessions     : '; grep -c '\"session_start\"' " ++ Util.shell_q(p) ++ "; " ++
```
